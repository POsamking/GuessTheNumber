# Guess The Number Game

Hey there, gamers! Welcome to my Guess The Number game, crafted with love using JavaScript. The goal is simple: guess the right number and win!

## How to Play

- You'll get a range of numbers to pick from (1-10).
- You have to try and guess the number one the search bar.
- If you are wrong, you will be informed with a red text and if you are right you will be informed with green text.
